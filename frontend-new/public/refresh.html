Refresh Application Cache

This page will help you clear your application cache and reload with the latest version.

If you're experiencing issues with the application not showing recent updates, this should help.

What this does:

  • Clears your browser's cache for this application
  • Reloads the application with the latest version
  • Redirects you to the home page

Manual refresh steps:

If the button above doesn't work, try these manual steps:

  1. Windows: Press Ctrl+F5 or Ctrl+Shift+R
  2. Mac: Press Cmd+Shift+R or Shift while clicking the reload button
  3. In some browsers, you can also open developer tools (F12) and disable cache in the Network tab
